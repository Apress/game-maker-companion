CD Contents

Welcome to the CD that accompanies The Game Maker's Companion. This CD will allow you to install the Game Maker tool, read extra documentation, and access all the resources and games for the book (plus a few more for you to experiment with).

For your reference this CD contains the following folders:

Program

This is where you will find the installation program for Game Maker (version 8.0) if you've not installed it already. Game Maker can be installed onto your computer by executing gmaker_inst.exe from this folder. This will add shortcuts for Game Maker into your Start menu so that you can run the program, read the documentation or uninstall it. 

The folder also contains a readme file which contains additional information and help for when a problem occurs. Also, please read the license agreement in this directory for using the Game Maker software.

Documents

This folder contains the full documentation for Game Maker in a number of different formats. Game Maker has many more features than the ones described in this book so see the documentation for more information.

ZOOL

The folders ranging from Chapter03 to Chapter06 contain all the files relevant to creating the game Zool. Zool and the Zool logo are registered trademarks of Urbanscan Ltd. All Rights Reserved.

The Zool game resources are provided solely for use with The Game Maker's Companion and should not be redistributed without written permission from the copyright holder.

FISHPOD and SHADOWS ON DECK

The Chapter02 folder and the folders ranging Chapter10 to Chapter13 contain all the files relevant to creating the games Fishpod and Shadows on Deck.

In purchasing this book, the authors and publisher grant you permission to use these electronic resources in your own Game Maker games, including for commercial use. However, redistribution of these resources is prohibited, as is their use for games not made with Game Maker.

Finished

This folder contains finished executables of all the games created in the book. We would advise copying them to your PC first for best performance. Note that the shadows.exe needs the shadows.wmv video file in order to play correctly.

More Information

You are strongly encouraged to check out the official Game Maker web site: www.yoyogames.com/make. Here you can download the most recent version of Game Maker, access user forums, find additional resources, and play many games created with Game Maker.
